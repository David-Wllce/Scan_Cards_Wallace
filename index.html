Importer vos contacts iPhone

Exportez vos contacts depuis l'iPhone en fichier .vcf, puis uploadez-le ici. L'import est intelligent : il détecte les doublons, lie automatiquement les contacts aux entreprises déjà dans votre CRM, et ne crée jamais deux fois le même contact.

Étape 1
Ouvrir Contacts
Sur iPhone : app Contacts → sélectionner les contacts → appuyer Partager → choisir Envoyer une copie vCard.
Étape 2
Récupérer le fichier
Envoyez le fichier .vcf par email, AirDrop sur votre Mac, ou iCloud Drive. Ouvrez-le ensuite sur ce navigateur.
Étape 3
Uploader ici
Déposez le fichier .vcf ci-dessous. L'import démarre automatiquement avec détection des doublons.

Automatisation

Raccourci iOS

Envoyez n'importe quel contact iPhone dans Notion en un seul tap — sans ouvrir l'app, sans scanner, sans copier-coller.

1
Notez votre URL Vercel

Renseignez-la dans Configuration (en haut). Elle ressemble à :

https://votre-projet.vercel.app/api/scan
2
Installer le Raccourci

Sur iPhone, ouvrez l'app Raccourcis et créez un nouveau raccourci avec les actions ci-contre. Ou scannez un QR code de partage si votre administrateur l'a configuré.

3
Configuration dans Raccourcis

Collez votre URL dans l'action Obtenir le contenu de l'URL. Corps de la requête JSON :

{   "action": "ios_shortcut",   "secret": "VOTRE_SECRET",   "contact": {     "prenom": [Prénom du contact],     "nom": [Nom de famille],     "email": [Adresses mail],     "telephone": [Numéros de téléphone],     "entreprise": [Société],     "poste": [Intitulé du poste]   } }

Le secret est la variable IOS_SHORTCUT_SECRET à définir dans Vercel.

4
Utiliser le raccourci

Dans l'app Contacts sur iPhone : ouvrez une fiche → faites défiler → Partager le contact → choisissez votre raccourci Wallace CRM. Le contact est instantanément dans Notion.

⚙️ Variables d'environnement Vercel
Ajoutez dans votre projet Vercel → Settings → Environment Variables :

IOS_SHORTCUT_SECRET = un mot de passe que vous choisissez (ex: wallace2024). Ce secret protège votre endpoint contre les appels non autorisés.
